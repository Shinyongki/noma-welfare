# 작업지시서_029_보충_조기TTS_상담신청구간_비활성화
# 작성일: 2026-03-15
# 목적: 상담 신청 흐름(이름·전화번호 수집)에서 조기 TTS 음성·화면 불일치 해결

---

## 문제

v029 첫 문장 조기 TTS 적용 후, 상담 신청 흐름에서 아래 불일치 발생.

- **화면**: "그럼 신동생님께 연락드릴 전화번호를 알려주시겠어요?"
- **음성**: 이전 턴 이름 요청 문구를 발화 중

**원인**: 스트리밍 초반에 이름 관련 문장이 첫 문장으로 감지되어 TTS가 조기 발화.
그 사이 Gemini는 계속 생성하여 전화번호 요청으로 응답이 완성됨.

---

## 해결 방향

상담 신청 흐름(이름·전화번호 수집 구간)에서는 조기 TTS를 스킵하고,
전체 응답 완료 후 기존 방식(speakText 전체 호출)으로 처리한다.

스킵 조건: 스트리밍 누적 텍스트에 아래 키워드 중 하나라도 포함되면 조기 TTS 비활성화.
- `"성함"`
- `"전화번호"`
- `"연락"`
- `"<noma-apply>"`

---

## 수정 내용

### code.html — 첫 문장 조기 TTS 스킵 조건 추가

**위치**: code.html의 첫 문장 조기 TTS 로직
(`firstTTSSent`, `streamAccumulated` 관련 코드 블록)

**현재 코드:**
```javascript
if (ttsEnabled && !firstTTSSent) {
  streamAccumulated += data.text;

  const sentenceEndMatch = streamAccumulated.match(/^(.{20,}[.?!])\s/);
  if (sentenceEndMatch) {
    firstTTSSent = true;
    const firstSentence = sentenceEndMatch[1];
    if (!firstSentence.includes('<noma-card>') && !firstSentence.includes('<noma-apply>')) {
      console.log('[TTS] 첫 문장 조기 호출:', firstSentence.slice(0, 30));
      firstTTSPromise = speakText(firstSentence);
    }
  }
}
```

**변경 코드:**
```javascript
if (ttsEnabled && !firstTTSSent) {
  streamAccumulated += data.text;

  // 상담 신청 흐름 감지 — 이름/전화번호 수집 구간에서는 조기 TTS 스킵
  const isConsultationFlow =
    streamAccumulated.includes('성함') ||
    streamAccumulated.includes('전화번호') ||
    streamAccumulated.includes('연락') ||
    streamAccumulated.includes('<noma-apply>');

  if (isConsultationFlow) {
    // 조기 TTS 비활성화 — 전체 응답 완료 후 기존 방식으로 처리
    firstTTSSent = true;  // 조기 TTS 재시도 방지
    firstTTSPromise = null;
  } else {
    const sentenceEndMatch = streamAccumulated.match(/^(.{20,}[.?!])\s/);
    if (sentenceEndMatch) {
      firstTTSSent = true;
      const firstSentence = sentenceEndMatch[1];
      if (!firstSentence.includes('<noma-card>') && !firstSentence.includes('<noma-apply>')) {
        console.log('[TTS] 첫 문장 조기 호출:', firstSentence.slice(0, 30));
        firstTTSPromise = speakText(firstSentence);
      }
    }
  }
}
```
