## 1. 나는 누구인가

경상남도사회서비스원 소속으로, **노마(Noma) AI 맞춤형 복지 내비게이터** 프로젝트를 Claude AI(채팅)와 Claude Code(로컬)를 함께 사용해서 개발하고 있다.

- **Claude AI (채팅, 지금 이 세션)**: 전략·방향·판단·문서 작성·작업지시서 생성
- **Claude Code (로컬 터미널)**: 실제 코드 수정·실행·커밋

---

## 2. 프로젝트 한 줄 요약

도민이 "몸이 아파요"처럼 일상어로 말하면 복지 서비스를 찾아주고, 상담 신청까지 연결되는 AI 복지 플랫폼. 내부 담당자는 상담 처리·부서 연계·협업 승인을 한 시스템에서 처리한다.

**배포 URL**: https://noma-welfare-production.up.railway.app/  
**리포**: Shinyongki/noma-welfare  
**스택**: Node.js/Express (ESM) + Gemini 2.0 Flash + TailwindCSS + Resend HTTP API

## 5. 프로젝트 핵심 구조 (빠른 참조)

```
stitch/code.html     ← 도민 검색·상담 화면
stitch/admin.html    ← 관리자 대시보드
stitch/case.html     ← 담당자 상담 처리
stitch/dept.html     ← 부서 조정자
server.js            ← Express 서버 단일 파일 (~3000줄, 라우트 포함)
data/requestStore.mjs ← 상담 신청 저장 (withLock 원자화 완료)
data/analyticsStore.mjs ← 분석 데이터 (withLock 원자화 완료)
```

**핵심 제약**: server.js 단일 파일 구조, withLock 필수, Resend HTTP API (SMTP 아님), Railway 포트 5000
